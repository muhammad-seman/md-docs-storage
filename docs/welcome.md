# Welcome to MD Docs Storage

This is your online markdown document storage.

## Features

- **Public Access**: Anyone can read documents
- **Admin Mode**: Create and edit with token
- **Simple**: Just markdown files
- **Free**: Powered by GitHub

## How to Use

### For Readers (Public)
1. Browse documents in the sidebar
2. Click to read
3. Download if needed

### For Admin
1. Click "Admin Mode"
2. Enter your GitHub token
3. Create new documents
4. Edit existing documents
5. Token is only stored in your browser session
